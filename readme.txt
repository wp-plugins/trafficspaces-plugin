== Description ==

The Trafficspaces Wordpress plugin helps you integrate ads easily into your Wordpress site.
It allows you to setup the layout of your ad zones in the sidebar without needing to edit any code. Once it is setup, your ads will start to display automatically.


For comments, questions and support use this email <a href="mailto:technical@trafficspaces.com">technical@trafficspaces.com</a>


== Installation ==

1. Download the plugin archive and extract the archive into your wp-content/plugins.
2. Go to the Plugins page in your WordPress Administration area and click 'Activate' for Trafficspaces Widgets.
3. Go to Widgets under Wordpress Appearance area and add Trafficspaces  Ad Zones to sidebar widgets.
4. On the Trafficspaces Ad Zones control panel, copy the ad zone id from your trafficspaces account ad tag code and paste in the Ad Zone text box accordingly.
